## ✨ Key Points & Improvements

### 1️⃣ 반응형 레이아웃 설계
- **Media Query**: 데스크탑/태블릿/모바일 3단계 브레이크포인트 설계
- 기존 별도 모바일 주소(`m.melon.com`) 방식을 단일 URL 반응형으로 통합

### 2️⃣ 음악 필터 인터랙션
- 장르/분위기별 필터링 UI 구현
- 필터 상태에 따른 콘텐츠 전환 처리

### 3️⃣ 팀 협업 경험
- **역할 분담**: 기획 단계부터 섹션별 담당 구분, 일관된 SCSS 변수 공유
- **Git 협업**: 커밋 컨벤션 준수 및 브랜치 전략 적용
